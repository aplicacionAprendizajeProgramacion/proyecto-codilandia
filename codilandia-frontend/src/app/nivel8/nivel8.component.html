🛤️NIVEL 8: CONDICIONALES IF, ELSE y ELSE IF🛤️

📚¿Y SI EL CAMINO SE DIVIDE?📚

A veces, los programas tienen que elegir qué hacer. Por ejemplo, si llueve, usamos paraguas ☔. Si hace sol, ¡vamos al parque! ☀️

Para esto usamos if, else y else if.

🚦 Ejemplo 1: Solo si pasa algo

Si tengo hambre, como una manzana 🍎

if (tengo_hambre) {{ '{' }}
   comer("manzana");
{{ '}' }}

🚥 Ejemplo 2: ¿Y si no sucede?

Si tengo frío, me pongo una chaqueta 🧥. Si no, voy en camiseta 👕

   poner("chaqueta");
{{ '}' }} else {{ '{' }}
   poner("camiseta");
{{ '}' }}
