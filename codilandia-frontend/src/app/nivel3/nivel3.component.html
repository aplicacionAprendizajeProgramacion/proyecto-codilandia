📚¿QUÉ ES IMPRIMIR EN PANTALLA?📚

En C++, usamos cout para mostrar cosas en pantalla.


Todo lo que esté entre comillas ("...") se muestra como texto literal.


Si quieres que el mensaje termine en una nueva línea, puedes añadir << endl;.




Por ejemplo:

cout << "Hola";

cout << "mundo";

🟡 Esto mostrará: Holamundo



cout << "Hola" << endl;

cout << "mundo";

🟢 Esto mostrará:
Hola
mundo

📝 Mostrar un mensaje 📝

Escribe un mensaje para mostrar en pantalla. Debe ir entre comillas dobles " ".


{{ frase1 }}

cout << {{ frase1 }} << endl; ❌
