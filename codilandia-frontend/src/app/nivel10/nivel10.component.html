🔁NIVEL 10: BUCLES FOR🔁

📚¿QUÉ ES UN BUCLE FOR?📚

A veces, queremos repetir algo un número exacto de veces. Para eso usamos for.

Por ejemplo:
👉 Contar del 1 al 5.
👉 Dar 10 saltos.

Ejemplo Notación:

for (int i = 0; i < 5; i++) {{ '{' }}
   salto();
{{ '}' }}

📚¿CÓMO FUNCIONA EL BUCLE FOR?📚

¡Vamos a entenderlo paso a paso!

  • 1️⃣ Empieza: Le damos un valor inicial a la variable.
    i = 0

  • 2️⃣ Revisa: Comprueba si la condición se cumple.
    i < 5

  • 3️⃣ Hace algo: Si la condición es verdadera, ejecuta el bloque de código.
    salto();

  • 4️⃣ Cambia: Al final, aumenta el valor de i.
    i++

    (esto es lo mismo que i = i + 1)


  • 5️⃣ Repite: Vuelve al paso 2 y repite hasta que la condición sea falsa.

  • 🏁 Termina: Cuando la condición no se cumple, el bucle se detiene y sigue el resto del programa.

¡Así es como contamos, saltamos o repetimos cosas con for! 🎉

⚠️¡CUIDADO CON LOS LÍMITES!⚠️

Si no configuras bien las condiciones del for, puedes hacer que no se ejecute nunca o que se ejecute demasiadas veces. 

for (int i = 10; i < 5; i++) {{ '{' }}
   ejemplo();
{{ '}' }}

Este bucle no se ejecuta porque i = 10 ya no cumple i < 5.

for (int i = 0; i < 1000; i++) {{ '{' }}
   ejemplo();
{{ '}' }}

¡Este bucle se ejecutará hasta 1000 veces! Son muchas iteraciones pero si la máquina lo puede soportar estamos seguro de que acabará.
