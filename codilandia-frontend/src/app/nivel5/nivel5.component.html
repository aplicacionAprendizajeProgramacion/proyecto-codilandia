🔍NIVEL 5: COMPARACIONES Y OPERADORES 🔍

📚¿QUÉ ES UN OPERADOR DE COMPARACIÓN?📚

Los usamos para comparar cosas. Por ejemplo, si tienes más manzanas que tu amigo > , ¡puedes decir que tienes más!

🔎 Tipos de operadores:

  • > Mayor que
  • < Menor que
  • >= Mayor o igual que
  • <= Menor o igual que
  • == Igual que
  • != Distinto de

¡En programación, nunca uses solo = para comparar!

int resultado = 5; ✅ → Esto guarda un valor.

resultado == 5 ✅ → Esto compara.




📝¿quién tiene más manzanas?📝

✅ ❌

int manzanas1 = 3;

int manzanas2 = 1;

manzanas1 > manzanas2



📝¿hay el mismo número de vacas?📝

✅ ❌

int vacas1 = 2;

int vacas2 = 1;

vacas1 == vacas2



📝¿hay menos trozos de tarta?📝

✅ ❌

int pastel1 = 2;

int pastel2 = 4;

pastel1 < pastel2



📝¿tiene al menos las mismas vacas?📝

int vacas1 = 2;

int vacas2 = 4;

¿Es cierto que vacas1 >= vacas2?

✅ ❌



📝¿tiene menos o igual pastel?📝

int pastel1 = 2;

int pastel2 = 2;

¿Es cierto que pastel1 <= pastel2?

✅ ❌



📝¿son distintos?📝

int a = 5;

¿Es cierto que a != b?

✅ ❌

📚¿SABÍAS QUE LAS COMPARACIONES TIENEN TAMBIÉN UN RESULTADO?📚

Cuando escribes una comparación como 2 > 8, ¡el ordenador te dice si es verdadero o falso! En este ejemplo, 2 > 8 es falso.

Como ese resultado es tan útil, lo puedes guardar en una variable. Recordemos que el tipo bool guarda "true" o "false". Por ejemplo:


int pelotas_rojas = 2;

int pelotas_azules = 8;

bool hay_mas_pelotas_rojas = (pelotas_rojas > pelotas_azules);


Ahora, la variable hay_mas_pelotas_rojas guarda el valor false porque pelotas_rojas es menor que pelotas_azules.


📝¿qué se guarda en la variable?📝

int edad1 = 8;

int edad2 = 8;

bool iguales = (edad1 == edad2);

¿Qué valor guarda la variable iguales?

✅ ❌

📚OPERADORES LÓGICOS📚

Los operadores lógicos combinan varias comparaciones y devuelven true (verdadero) o false (falso). En la mayoría de lenguajes usamos:

  • && (AND) – Verdadero si todas las condiciones son verdaderas.
  • || (OR) – Verdadero si al menos una es verdadera.
  • ! (NOT) – Invierte el resultado de una condición.




📝 ¿puede Juan salir al recreo(AND) 📝

bool ha_terminado_tarea = true;

bool tiene_permiso = false;

bool puede_salir = (ha_terminado_tarea && tiene_permiso);

¿Qué valor tendrá puede_salir?

✅ ❌




📝 ¿puede Inés ver una peli con sus padres? (OR) 📝

bool es_fin_de_semana = false;

bool ha_terminado_los_deberes = true;

bool puede_ver_peli = (es_fin_de_semana || ha_terminado_los_deberes);

¿Qué valor tendrá puede_ver_peli?

✅ ❌




📝 ¿está silencioso? (NOT) 📝

bool ruido = true;

bool silencioso = !ruido;

¿Qué valor tendrá silencioso?

✅ ❌
