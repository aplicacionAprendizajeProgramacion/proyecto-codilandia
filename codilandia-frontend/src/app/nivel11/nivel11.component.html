📚¿QUÉ ES UNA FUNCIÓN?📚

Una función en programación es como una receta o una orden mágica que le damos al ordenador. 🧙‍♂️💡
Sirve para repetir instrucciones sin tener que escribirlas muchas veces.

Por ejemplo, si queremos que el ordenador diga "¡Hola Codilandia!", podemos escribirlo dentro de una función.

tipo nombre_funcion(parámetros) {{ '{' }}
// instrucciones
{{ '}' }}

🔍 ¿Qué significa cada parte?

  • tipo → qué devuelve la función: puede ser int, string, bool, double... o void si no devuelve nada.

  • nombre_funcion → el nombre que le damos (¡el que usaremos para llamarla!)

  • parámetros → lo que necesita para funcionar (puede estar vacío)

  • return → si el tipo NO es void, usamos return para decir qué valor devuelve.

  • {{ '{' }} instrucciones {{ '}' }} → lo que hace la función, como si fuera su cuerpo



Ejemplo 1: Esta función void saludar() no devuelve nada (no hace return) ya que es de tipo void, y solo imprime un saludo:

void saludar() {{ '{' }}

cout << "¡Hola Codilandia!" << endl;

{{ '}' }}




🧮 También podemos tener funciones que devuelven valores, como un número.

int sumar() {{ '{' }}

return 2 + 3;

{{ '}' }}

Esta función se llama sumar() y devuelve un número (int) porque 2 + 3 = 5.



✅ También podemos usar bool para funciones que devuelven verdadero o falso:

bool esMayor() {{ '{' }}

return 10 > 5;

{{ '}' }}

Aquí devolvemos true porque 10 sí es mayor que 5.





✉️ Y también podemos devolver texto usando string:

string getNombre() {{ '{' }}

return "Leo";

{{ '}' }}

Esto devuelve la palabra "Leo". ¡Como si le preguntaras al ordenador: “¿Cómo te llamas?”!





🧮 Por último, si queremos un número con decimales, usamos double:

double calcularArea() {{ '{' }}

return 3.14 * 2 * 2;

{{ '}' }}

¡Esta función devuelve el área de un círculo de radio 2!

Queremos una función que diga "¡Bienvenido a Codilandia!".
Rellena los espacios:

____ bienvenida() {{ '{' }}
cout << "¡Bienvenido a Codilandia!";
{{ '}' }}

🛠️ ¿CÓMO LLAMAMOS A UNA FUNCIÓN? 🛠️

Una vez que hemos escrito nuestra función, como por ejemplo esta que saluda:

void saludar() {{ '{' }}

cout << "¡Hola, Codilandia!" << endl;

{{ '}' }}

Todavía no se ejecuta sola. 🤔 Para que ocurra lo que hay dentro, hay que llamarla desde otra parte del programa, como main().

👇 Así lo hacemos:

int main() {{ '{' }}

saludar();

return 0;

{{ '}' }}

Fíjate que la escribimos con su nombre y (). ¡Eso le dice al programa: “ejecuta lo que hay dentro”! 🔁




Entonces, si tenemos esta función:

void bienvenida() {{ '{' }}

cout << "¡Bienvenido a Codilandia!" << endl;

{{ '}' }}

👉 ¿Cómo la llamarías tú para que funcione?


✅ ❌

bienvenida();

🎁 ¿QUÉ SON LOS PARÁMETROS?

Los parámetros son como ingredientes que necesita una receta. 🍰📦
Si hacemos una función que sume, necesita los dos números que queremos sumar. Esos números son sus parámetros.

int sumar(int a, int b) {{ '{' }}

return a + b;

{{ '}' }}

Aquí la función se llama sumar y necesita a y b para funcionar.
Cuando la usamos así: sumar(3, 5), devuelve 8 ✅

También podemos tener funciones como esta, que saludan usando un string:

void saludar(string nombre) {{ '{' }}

cout << "¡Hola " << nombre << "!";

{{ '}' }}

Si ponemos saludar("Ana"), el resultado sería:
¡Hola Ana!

🧪 Escribe los parámetros correctos

Queremos una función que reciba dos números enteros para sumarlos.
Escribe los parámetros correctos que irían dentro de los paréntesis.

👀 Recuerda que cada parámetro empieza por tipo de variable y debe tener un nombre. Los parámetros se separan por comas
