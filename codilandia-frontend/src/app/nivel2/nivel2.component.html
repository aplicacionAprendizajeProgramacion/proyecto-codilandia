📚¿QUÉ ES UNA VARIABLE?📚

Una variable es como una cajita donde guardamos información en un programa. Le damos un nombre, y dentro metemos un valor. Las variables se usan mucho para que el ordenador recuerde datos como números, frases, o si algo es verdadero o falso.

✍️ En programación, las variables se escriben así:

tipo nombre_variable = valor;

Es importante que el nombre de una variable represente lo que se quiere guardar (porque así es más fácil recordar luego😉).

Además, para crear el nombre de una variable, es importante no poner espacios, si se quiere poner más de una palabra, se puede poner uniéndolo con guiones como: platos_arroz.

Una variable siempre debe terminar en ";" al declararse (que es la forma en la que se da valor a una variable).

Por ejemplo: int edad = 9;

📚DOUBLE (REAL)📚

¿Qué es un número real?

Es un número que puede tener decimales, como 1.50 o 2.55

Ejemplo: Si usas una moneda de 2 €, otra de 1 € y una de 50 céntimos (0.50 €), juntas hacen 3.50 €.




📝selecciona (pulsando) monedas para sumar 2,55 € :📝

✏️ Así lo escribimos: double monedas


✖

Total: {{ sumaMonedas.toFixed(2) }} €

✅ ❌ double monedas = {{ sumaMonedas.toFixed(2) }};

📚BOOLEANO📚

¿Qué es un booleano?

Es una variable que solo puede tener dos valores: true (verdadero) o false (falso).

Ejemplo: ¿Hoy he desayunado? Si sí, entonces es true, si no, es false.

✏️ Así lo escribimos: bool he_desayunado = true;




📝¿hay un pastel?📝

✅ ❌ bool hay_pastel = true;



📝¿hay tres trozos de pastel?📝

✅ ❌ bool hay_tres_trozos = false;

📚STRING📚

¿Qué es un string?

Es un tipo de variable que se usa para guardar palabras o frases. Como cuando escribes tu nombre o una historia.

Ejemplo: "Hola, me llamo Leo"

✏️ Así lo escribimos: string texto
