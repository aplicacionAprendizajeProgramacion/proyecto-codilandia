📚¿QUÉ PASA SI MEZCLAMOS UN NÚMERO CON DECIMALES Y UNO SIN DECIMALES?📚

Imagina que tienes dos cajas:

  • 🟫 Caja entera (int): solo guarda números sin decimales, como 1, 2, 3...
  • 🟧 Caja decimal (double): guarda todo, incluso 2.5, 7.25 o 0.1.

¿Qué pasa si haces una operación entre los dos?

Siempre gana la caja grande (double) porque la caja int es demasiado pequeña para guardar decimales.

🍩 Ejemplo:

double numero1 = 5.0;
int numero2 = 2;

double resultado = numero1 / numero2;

✅ Se guarda 2.5 correctamente

❌ ¿Y si usamos int?

int resultado = numero1 / numero2;

⚠️ Solo se guarda 2, se pierde el .5


¿Cuándo pasa esto?

Esto ocurre siempre que usas double e int juntos, sin importar si haces:

  • ➕ Sumas
  • ➖ Restas
  • ✖️ Multiplicaciones
  • ➗ Divisiones

✅ Usa double si quieres guardar el resultado completo con decimales.
